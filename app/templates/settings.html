Auto Sync Schedule
To run daily auto-sync, add a cron job (Linux/macOS) or Scheduled Task (Windows): 
# Runs every day at 02:00
0 2 * * * curl -s -X POST http://localhost:5000/crawler/api/start \
  -H "Content-Type: application/json" \
  -d '{"mode":"auto_sync"}'
Auto Sync covers the last 2 calendar months up to today, split into 30-day chunks automatically.
